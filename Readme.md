# Drools Rules Liferay Knowledge Base Plugin

Plugin for Drools Engine that grants access to Liferay to read Articles texts

## Usage

As this is a maven `pom.xml` file, the way to use it is to define it as a parent in your project.

```
<parent>
	<groupId>com.biit-solutions.drools.plugins</groupId>
	<artifactId>liferay-knowledge-base</artifactId>
	<packaging>jar</packaging>
</parent>
```
